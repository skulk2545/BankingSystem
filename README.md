# 🏦 BankingSystem

## **Overview**

The **BankingSystem** project is a simple Java-based simulation of bank operations like:

- Deposit  
- Withdraw  
- Check Balance

This project is built using **Object-Oriented Programming (OOPS)** concepts such as:

- **Interfaces**
- **Abstract Classes**
- **Inheritance**
- **Polymorphism**
